# SuperVehicle
This is a private project on Nvidia Jetson TX2.

项目构成：

| preproc |SSCA | grid | utils | test |
|------------|-----------|--------|--------|---------|
| 预处理（包括基线矫正，图片处理等） | 深度测距 | 栅格地图构建 | I/O以及其他便捷函数 | 测试|
